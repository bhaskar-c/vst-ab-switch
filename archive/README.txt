Install instructions
====================

macOS
-----
For VST2, copy ABSwitch.vst under $HOME/Library/Audio/Plug-ins/VST for a single user or /Library/Audio/Plug-ins/VST for all users (may require admin access)
For VST2, copy ABSwitch.vst3 under $HOME/Library/Audio/Plug-ins/VST3 for a single user or /Library/Audio/Plug-ins/VST3 for all users (may require admin access)

Usage
=====
Check the documentation at https://pongasoft.com/rack-extensions/ABSwitch.html
